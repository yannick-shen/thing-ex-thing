# 页面导航说明

## 📱 页面访问入口

### 1. 政策页面（用户协议 / 隐私政策 / 平台规则）
**访问方式：**
- 从"我的发布"页面 → 点击"设置"按钮 → 点击"用户协议"、"隐私政策"或"平台规则"
- 路径：`我的发布` → `设置` → `用户协议` / `隐私政策` / `平台规则`

**功能：**
- 用户协议
- 隐私政策
- 平台规则

---

### 2. 设置页面
**访问方式：**
- 从"我的发布"页面 → 点击右上角"设置"按钮
- 路径：`我的发布` → `设置`

**功能：**
- 账号信息（个人信息、绑定手机号）
- 功能设置（消息通知、位置权限）
- 帮助与反馈（用户协议、平台规则、隐私政策、联系客服、意见反馈）
- 关于（版本号）
- 退出登录

---

### 3. 消息页面
**访问方式：**
- 从"我的发布"页面 → 点击右上角"消息"按钮
- 路径：`我的发布` → `消息`

**功能：**
- 评论消息（待接入数据）
- 系统消息（待接入数据）

---

## 🗺️ 页面导航结构

```
首页（地图）
  ├── 发布页面（发布按钮）
  └── 详情页面（点击地图标记）

我的发布（TabBar）
  ├── 设置按钮 → 设置页面
  │   ├── 平台规则 → 政策页面
  │   └── 隐私政策 → 政策页面
  └── 消息按钮 → 消息页面

收藏（TabBar）
  └── 详情页面（点击收藏物品）
```

## 🔧 需要执行的操作

### 1. 重新编译小程序
在微信开发者工具中：
- 点击"编译"按钮
- 等待编译完成

### 2. 测试页面导航
1. 进入"我的发布"页面（底部TabBar）
2. 查看右上角是否有"消息"和"设置"按钮
3. 点击"设置"按钮，应该能进入设置页面
4. 在设置页面点击"用户协议"、"隐私政策"或"平台规则"，应该能进入对应的政策页面
5. 返回"我的发布"页面，点击"消息"按钮，应该能进入消息页面
6. 首次登录或清除缓存后，打开首页会弹出《用户协议》《隐私政策》《平台规则》确认弹窗，必须同意才能继续使用
